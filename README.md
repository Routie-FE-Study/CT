# 루티 코테 연습장 

## 기본 규칙
1. 매주 월요일에 이번 주 목표를 올린다.
2. 매주 금요일 23:50까지 목표를 달성하지 못하면 벌금 1,000원을 납부한다.
3. 주 목표 최소치는 3회로 제한한다.

## 제출 규칙
1. 문제 유형을 명시한다.
2. 풀이 시간을 명시한다.
3. 접근 방식을 명시한다.

## 파일 생성 규칙
1. year.month(week) 형식으로 폴더를 생성한다. `26.01(1주차)`
2. day(order -> 이 날의 몇 번째 풀이인지).js 형식으로 파일을 생성한다. `07(1).js`
3. 풀이 파일과 같은 이름의 md 파일을 만들어 풀이에 대한 내용을 작성한다. `TEMPLATE.md` 참고
