# 접미사 배열
- 문제 링크: [여기](https://www.acmicpc.net/problem/11656)
- 난이도: s4
## 🔖 문제 유형
- 정렬
- 문자열
## ⏰ 풀이 시간(min) 
20 min

## 🤔 스스로 고민한 시간(min) 
20 min

## 🧰 접근 방법

- 입력 문자열을 문자 별로 분리(배열)
- 접미사를 모아둘 배열을 초기화
- 문자 배열의 길이만큼 for문으로 순회하며 `slice(i).join('')`로 접미사 생성
- 접미사를 모을 배열에 추가
- 문자열 정렬로 배열 순서를 정렬하고 forEach로 console.log 실행
